registerHandler

  • registerHandler<S, R>(handlerName: string, options?: Partial<RegisterHandlerOptions<S, R>>): this
    • Defined in Bridge/DesktopBridge.ts:175

    注册一个Handler, 用于向Bridge 发送消息

    Type parameters

    • S

    • R

    Parameters

    • handlerName: string

      Handler 名

    • Default value options: Partial<RegisterHandlerOptions<S, R>> = {}

      选项

      • beforeSend: 在发送之前对数据进行预处理, 默认使用JSON.stringify转换为字符串，注 意Bridge只接受字符串
      • afterReceived: 在调用回调之前，对Bridge返回的数据进行转换
      • preserveName: 是否对HandlerName进行转换，按照规范转换为 On形式， 不过一 些接口没有安装规范来, 通过设置这个接口为true，避免转换HandlerName， 默认为false

    Returns this
